# VS-Chrome 

High-performance browser integration for Visual Studio Code.

## Features
- Integrated Sidebar Browser
- Smart URL/Search bar (Google Search integration)
- Persistent Session State
- Quick-access Refresh & Clear History commands
- Native VS Code Theme adaptation

## Setup
1. `npm install`
2. `npm run compile`
3. Press `F5` to debug
